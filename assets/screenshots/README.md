# Screenshot per App Store

Questa cartella contiene gli screenshot necessari per la pubblicazione sugli store.

## DIMENSIONI RICHIESTE

### iOS (Apple App Store)
- **iPhone 6.5"**: 1284 x 2778 px (o 1242 x 2688 px)
- **iPhone 5.5"**: 1242 x 2208 px
- **iPad Pro 12.9"**: 2048 x 2732 px (opzionale)

### Android (Google Play)
- **Telefono**: Minimo 320px, massimo 3840px (16:9 o 9:16)
- **Tablet 7"**: Opzionale
- **Tablet 10"**: Opzionale

## SCREENSHOT CONSIGLIATI (5 per piattaforma)

1. **Main Menu** - Schermata principale con bottoni gioco
2. **Gameplay Combattimento** - Azione durante una battaglia
3. **Mappa/Esplorazione** - Vista della mappa con percorsi
4. **Collezione Carte** - Griglia carte con rarità
5. **Achievement/Statistiche** - Progressi giocatore

## LINEE GUIDA

- NON includere testo promozionale negli screenshot
- Mostrare contenuto reale del gameplay
- Usare momenti d'azione dinamici
- Evitare schermate di caricamento
- Assicurarsi che il testo nell'app sia leggibile

## STRUMENTI CONSIGLIATI

- Xcode Simulator (iOS)
- Android Studio Emulator
- Device Farm reali
- Strumenti di screenshot nativi

## NOTE

Gli screenshot devono essere aggiornati ad ogni cambio significativo dell'UI.
